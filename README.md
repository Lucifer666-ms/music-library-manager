# Music Library Manager

A music management system developed using Java and SQL for storing and managing music records.

## Technologies Used

* Java
* SQL
* HTML
* CSS
* JavaScript

## Features

* Add new songs
* Update song records
* Delete songs
* Search and manage music library
* Database integration using SQL

## Learning Outcomes

* Object-Oriented Programming (OOP)
* Database Management
* CRUD Operations
* Java Application Development
